## Instalasi

Langkah-langkah untuk menginstal proyek ini:

1. Clone repositori ini.
   bash
   git clone git@github.com:ihsanzakyf/VueJs-3.git
   
2. Masuk ke direktori proyek.
   bash
   cd nama-proyek
   
3. Instal dependensi
   bash
   npm install

## Penggunaan

Berikut langkah-langkah untuk menjalankan proyek ini:

1. Jalankan Server.
    ```bash
    npm run dev
    
2. Buka di browser.
   ```bash
   http://localhost:xxxx
